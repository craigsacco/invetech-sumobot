Install Linux Mint 17.1 Cinnamon (with defaults)

Update all packages from APT
 * sudo apt-get update
 * sudo apt-get dist-upgrade

Install VMware Tools
 > select "Install VMware Tools" from the VMWare menu
 * cd ~/Desktop
 * tar xvfp /media/developer/VMware\ Tools/VMwareTools-*.tar.gz
 * pushd vmware-tools-distrib
 * sudo ./vmware-install.pl
 * popd
 * sudo rm -rf vmware-tools-distrib

Install additional developement packages
 * sudo apt-get install build-essential bison gawk make automake autoconf gcc g++ gcc-multilib g++-multilib gcc-doc flex autogen libtool binutils m4 expect

Install source control tools
 * sudo apt-get install git gitk

Install GNU ARM development toolchain
 * sudo add-apt-repository ppa:terry.guo/gcc-arm-embedded
 * sudo apt-get update
 * sudo apt-get install gcc-arm-none-eabi=4.9.3.2014q4-0trusty12

Install hardware debugger
 * sudo apt-get install openocd
